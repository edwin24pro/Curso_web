Archivo extorno

El objetivo de crear mi archivo estilos.css es para separar al estructuradel diseño.

  • Asi tendremos mi archivo .html solo para la estructura elementos etiquetas y contenido.

    • Y mi archivo .css solo para los selectores y propiedades de diseño que le asignare a cada etiqueta.

Modo de uso

Para relacinar un archivo html con su archivo css (hojas de estilo). Debemos hacer uso de nuestra etiqueta link en la cabecera del documento. y en su atributo href le indicamos la ruta donde se encuentra el archivo.
